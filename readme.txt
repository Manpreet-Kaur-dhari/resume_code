# One-Page Resume Webpage Project

# Manpreet Kaur - SOC Analyst Intern Resume

This project contains a one-page resume webpage for Manpreet Kaur, a SOC Analyst Intern. The webpage is designed using HTML and CSS, and it includes information about Manpreet's professional background, education, work experience, skills, and contact information..

## Project Requirements
## File Structure

- Resume.html: Main HTML file containing the structure of the resume webpage.
- css/new.css: External CSS file for styling the webpage.
- js/script.js: JavaScript file for handling the download PDF functionality using html2pdf library.And I include javascript inside the main file.
- img/: Folder containing images, such as the user's profile picture.
- README.txt: This file.

### Structure
The webpage includes the following sections:

1. Header:
   - Your name.
   - A brief introduction or professional summary.

2. About Me:
   - A concise overview of your background, skills, and expertise.

3. Education:
   - List of educational qualifications, including degrees, institutions, and graduation years.

4. Work Experience:
   - Highlighting relevant work experience, including job titles, companies, and job responsibilities.

5. Skills:
   - Showcase of technical skills, soft skills, and any certifications or awards.

6. Contact Information:
   - Email address.
   - Phone number.
   - Optionally, links to your LinkedIn profile or portfolio website.
